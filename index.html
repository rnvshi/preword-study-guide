HTML

  • HTML elements build the structure of a webpage and provide framework elements for the user to navigate the webpage.
  • The head element contains metadata about the webpage. These elements are not visible to the user.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles associated with the website.
  • The base element defines the base URL for the webpage.
  • The link element connect an external resource to the webpage.
  • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
